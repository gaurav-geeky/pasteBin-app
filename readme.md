# Pastebin Lite

## Description
Pastebin Lite is a minimal full-stack web application that allows users to create text pastes and share them through a unique URL.  
Each paste can optionally expire based on time-to-live (TTL) or a maximum number of views, after which it becomes unavailable.

## Run Locally

### Backend
```bash
cd backend
npm install
node server.js


### Frontend 

cd frontend
pnpm install
pnpm run dev
